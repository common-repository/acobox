== Description ==

Acobox.com eliminates the need to download or upload files when using pictures and images. Now with Acobox Plugin 2.0 for WordPress, you don't even have to search for pictures or images:

After you install the plugin, you will see an Acobox section on the New Post page. Just type the blog post as usual, and then the pictures matching your post topic will appear automatically. Click on a thumbnail you can access the image code as well as options like Relevant Images.

Yes, the Acobox plugin is designed for 'lazy' bloggers who even don't bother to search!

See also http://acobox.com/content/about

== Installation ==

1. Upload folder `acobox` and its files to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
